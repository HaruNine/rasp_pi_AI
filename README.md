# rasp_pi_AI
라즈베리환경에서 만든 파이썬기반 대화형 ai
raspberry pi4 mode B and dlpdlcr2000evm
## 활용 api
google cloud의 stt, tts
, chatGPT API

## 환경
라즈베리파이 + dlpdlcr2000evm 을활용하여 대화형 홀로그램재작
